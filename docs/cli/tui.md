# `Root tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
Root tui
Root tui --url ws://127.0.0.1:18789 --token <token>
Root tui --session main --deliver
```
