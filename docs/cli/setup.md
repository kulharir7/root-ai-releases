# `Root setup`

Initialize `~/.Root/Root.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
Root setup
Root setup --workspace ~/.Root/workspace
```

To run the wizard via setup:

```bash
Root setup --wizard
```
